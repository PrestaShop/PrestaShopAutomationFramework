public native find($selector, $options = array())

Find element(s).

Selectors are interpreted as CSS by default, example: ".main ul li"

To use an xpath selector, prefix it with {xpath}, for instance "{xpath}}//div".

The options you can provide are: - unique: boolean, defaults to true - whether or not the element you're looking for is unique - wait: boolean, defaults to true - wait for the element to show up if it is not immediately there
